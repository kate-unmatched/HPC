# Лабораторные работы по предмету "Высокопроизводительные вычисления"

### [Лабораторная работа №1. Перемножение матриц](Lab_1_HPC)

В рамках лабораторной работы реализовано умножение квадратных матриц на CPU и GPU с использованием технологии CUDA. Параллелизация выполнена на уровне элементов результирующей матрицы, где каждый поток GPU вычисляет отдельный элемент для эффективного использования вычислительных ресурсов. Проведённые эксперименты показали значительное ускорение на GPU по сравнению с CPU при увеличении размера матриц. 

Файл README содержится в папке с лабораторной работой.
